# Smart Soil — AI-Powered Soil Health Analyzer 🚀

Smart Soil is a **full-stack Android app** for Indian farmers to test soil health,
get crop recommendations, and chat with AI about their fields.

## 📱 Features
✅ **Auth** — Register/Login  
✅ **Farm Management** — Add/Edit/Delete farms  
✅ **Soil Testing** — Camera/Gallery upload + Image validation  
✅ **Soil Prediction** — SOC, N, P, K, pH analysis  
✅ **History** — Farm-wise test history  
✅ **Custom Navigation** — Bottom nav bar  
✅ **Image Standardization** — Consistent ML input
